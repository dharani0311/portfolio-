Dharani Portfolio Project Documentation

Project Overview
This portfolio website showcases Dharani Rachamreddy’s web development skills, projects, and contact information. It is designed to be modern, visually appealing, and fully responsive across desktop, tablet, and mobile devices.

1. Website Structure
Navigation Bar:
oIncludes a logo for branding and a menu with links to sections: Home, About, Skills, Projects, and Contact.
oFeatures a hamburger menu for mobile devices.
Home Section:
oDisplays a profile image alongside introduction text, name, role, and description.
oProvides call-to-action buttons for downloading a CV and contacting directly.
About Section:
oIntroduces personal and professional background.
oHighlights web development philosophy and interests.
Skills Section:
oLists technical skills with visual proficiency bars to indicate level of expertise.
oSkills include HTML, CSS, JavaScript, UI basics, and responsive design.
Projects Section:
oHighlights selected projects with titles and brief descriptions.
oDemonstrates practical web development experience.
Contact Section:
oProvides email and phone number for direct communication.
oEncourages visitors to reach out for collaboration or inquiries.

2. Styling and Layout
Global Styling:
oConsistent font and spacing across all sections.
oDefault margin and padding removed for uniformity.
Navigation Bar:
oFlexbox layout for horizontal alignment of logo and menu items.
oSticky position ensures visibility while scrolling.
oInteractive hover effects on links.
Home Section:
oProfile image displayed in circular shape.
oIntroduction text styled for emphasis, readability, and hierarchy.
oButtons styled for visual prominence and hover interactivity.
Sections:
oConsistent padding and spacing for a clean layout.
oHeadings highlighted with color and font size.
oParagraphs formatted for readability with line spacing.
Skills Visualization:
oProgress bars represent skill proficiency.
oEach skill has a distinct width corresponding to expertise.
Project Cards:
oVisually distinct cards with background color and border accent.
oStructured layout for clarity and readability.
Contact Section:
oText formatted with sufficient font size and spacing.
oLinks for email and phone are interactive.
Background Effects:
oFixed background image with blur effect for modern aesthetic.
oPositioned behind all content to avoid interference with readability.

3. Responsive Design
Tablet and Mobile:
oNavigation menu transforms into a toggleable hamburger menu.
oSections stack vertically for small screens.
oImages, headings, text, and buttons resize appropriately.
oPadding and gaps adjusted to maintain visual balance.
Extra Small Screens:
oFurther reduction in image size and font sizes.
oEnsures all content remains readable and accessible.

4. Key Features
Fully responsive design across all devices.
Sticky, interactive navigation bar with hamburger toggle.
Animated hover effects on buttons and links.
Skill proficiency bars for visual representation of expertise.
Modern aesthetic with circular profile image and blurred background.
Organized layout of projects and personal information for easy navigation.

5. Usage
The website is designed to be easily customizable by updating text, images, and links.
Suitable for showcasing personal portfolio, professional skills, and projects.
